## Status: ✅ Complete

## Commands

### `strata init`

Idempotent project scaffolding. Creates:
- `components/atoms/_contract.yml`
- `components/molecules/_contract.yml`
- `components/organisms/_contract.yml`
- `routes/_contract.yml`
- `.claude/skills/STRATA-SKILL.md`
- Adds `strata:check` and `strata:build` scripts to `package.json`

Second run prints "Already exists" for each item. Never overwrites.

### `strata check`

Finds project root via `findProjectRoot()`, runs contract walker + validator, prints coloured diagnostics. Exit code 1 on errors, 0 on clean (warnings don't fail).

### `strata build`

Walks contract tree, generates `.strata.types.ts` files next to each route `_contract.yml` that has `layout.slots`. Prints count of files generated.

### `strata add`

Subcommands: `route`, `atom`, `molecule`, `organism`.

Route flags: `--fills`, `--slots`, `--default`, `--redirect`, `--param`

Auto-detects `param` from bracket segments (`[fundId]` → `param: fundId`). Runs `strata build` after scaffolding.

## Root Detection

`findProjectRoot()` walks up from `process.cwd()` checking:
1. `package.json` with strata dependency (most specific)
2. `pnpm-workspace.yaml` (monorepo fallback)
3. Falls back to cwd

Shared by `check`, `build`, and `add` commands.

## Type Name Generation

`pathToTypeName(relativePath)`:
1. Relative path from `routes/` directory
2. Split into segments
3. Strip `[` `]` brackets
4. PascalCase each segment
5. Concatenate

| Path | Type Name |
|------|-----------|
| `routes/` | `Root` |
| `routes/funds` | `Funds` |
| `routes/funds/[fundId]` | `FundsFundId` |
| `routes/fund-detail` | `FundDetail` |
| `routes/funds/[fundId]/holdings/[holdingId]` | `FundsFundIdHoldingsHoldingId` |

Generated type file example:
```ts
// AUTO-GENERATED by strata build — do not edit
export type FundsFundIdSlots = 'tabs' | 'contextPanel';
export type FundsFundIdRequiredSlots = 'tabs';
export type FundsFundIdInheritedSlots = 'breadcrumb' | 'actions';
export type FundsFundIdAllSlots = FundsFundIdSlots | FundsFundIdInheritedSlots;
```

Empty categories use `never` (e.g. `type RootRequiredSlots = never;`).

## Key Decisions

| Decision | Rationale | Watch? |
|----------|-----------|--------|
| STRATA-SKILL.md embedded as string in `skill-content.ts` | Bundled with package, no file resolution needed at runtime | ⚠️ Means updates to docs/STRATA-SKILL.md need manual sync to template |
| Auto-detect param from brackets | `[fundId]` → `param: fundId` without explicit flag | ✅ Ergonomic |
| `--default` and `--redirect` mutual exclusivity via validation, not arg parsing | Simpler CLI code, validator already has the rule | ✅ Good — single source of truth |
| `strata add` runs `strata build` after scaffolding | New route gets types immediately | ✅ Good DX |
| `never` for empty type categories | `type RootRequiredSlots = never` when no required slots | ✅ Type-safe |
